## Tips for Best Results

1. **Seamless Tiling**: For floor and ceiling tiles, add "seamless" or "tileable" to ensure edges match
2. **Consistent Lighting**: Specify "neutral lighting" or "evenly lit" to let the game engine handle lighting
3. **No Strong Shadows**: Avoid prompts that create strong directional shadows
4. **Downscale Quality**: Generate at 256x256 or 512x512, then downscale to 64x64 for crisp results
5. **Batch Generation**: Generate multiple variations and pick the best ones
6. **Post-Processing**: Use image editors to adjust contrast, remove artifacts, ensure seamless tiling

## Negative Prompts (for Stable Diffusion)

Add these to avoid common issues:
```
text, watermark, signature, blurry, low quality, distorted, 3D render, photo, realistic, human, face, character, UI elements, border, frame
```
